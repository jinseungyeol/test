기본기가 잘 되어있는 신입 웹 퍼블리셔 ‘진승열’ 입니다. 주석처리는 섹션마다 상세히 기입하고 있고 인덴트가 어긋나지 않게 퍼블리싱 하고 있으며 images, plugin 등 주제별로 폴더구조를 나누고 그 안에 파일명, 폴더명 또한 잘 정리해 놓았습니다. 또 저는 기능성을 제외하고는 플러그인에 의존하지 않고 하드코딩을 원칙으로 제작합니다. 탭 메뉴 제작 시 순수 CSS로 제작이 가능하며 이외에도 어코디언, 간단한 애니메이션은 플러그인을 사용하지 않고 하드코딩으로 제작이 가능합니다.

Skills

HTML

웹 표준과 웹 접근성을 준수하는 HTML 정보 구조 설계로 웹사이트 퍼블리싱을 해오고 있습니다. 특히 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다.

CSS

CSS 코딩은 IE 지원도 충분히 고려하는 크로스 브라우징 (Cross Browsing)을 위해 안정적 CSS 속성을 사용해서 HTML Elements를 배치하고 디자인합니다. 특별한 경우 CSS3에서 새롭게 지원하는 속성을 활용해 더 다양한 CSS 배치와 디자인을 합니다.

JQUERY

내비게이션, 탭 메뉴, 어코디언 등 웹 퍼블리싱에 반드시 필요한 상호작용을 위한 제이쿼리는 플러그인을 사용하지 않고 순수한 제이쿼리 코딩으로 제작합니다.

RESPONSIVE WEB

반응형 웹 제작에 있어서 크로스 브라우징 (Cross Browsing)을 위해 기존 방식인 float, position, Media Query를 활용한 반응형 웹 제작 경험을 가지고 있습니다. 대부분 기존 방식으로 제작하지만, 모바일 웹 앱 제작의 경우 Flex와 Grid를 활용하여 제작합니다.

MOBILE WEB APP

모바일 우선을 위해 Mobile First에 준하는 모바일 웹 앱 제작 경험도 가지고 있습니다. UI/UX 측면에서 미적인 디자인뿐만 아니라 모바일 사용자 경험에 최적화를 위해 효율적인 콘텐츠 배치를 우선으로 생각합니다.

PHOTOSHOP

웹 사이트 UI 디자인 작업은 포토샵을 활용하며 웹 UI 디자인 작업을 위한 포토샵 사용 능력은 충분하다고 생각합니다.

텍스트 에디터는 Visual Studio Code를 메인으로 사용하며 때에 따라 Adobe Brackets를 병행 사용하고 간단한 부분 수정을 위해 EditPlus도 활용합니다.

Basic Principles.

사용자 입장에서 쉽게 접근할 수 있는 웹 구축을 위해 팀원들과 소통하며 구조화된 마크업으로 개발하고 있습니다. 저에게 코딩이란 부지런하게 삶을 살 수 있게 하는 목표와도 같으며 잘하고 싶어 하는 일이기도 합니다.

웹 퍼블리셔라는 새로운 시작을 하게 되었습니다. 한발 더 나아가기 위해 끊임없이 노력하며 일은 혼자 하는 것이 아니기에, 늘 배우고 탐구하는 자세로 소통과 기획능력까지 갖춘 웹 퍼블리셔가 되겠습니다.
